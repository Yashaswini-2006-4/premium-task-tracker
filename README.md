# вҡҷпёҸ Installation

## Clone Repository

```bash
git clone https://github.com/Yashaswini-2006-4/premium-task-tracker.git
```

---

## Backend Setup

```bash
cd server
npm install
```

Create a **.env** file inside the server folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string
```

Run Backend

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on

```
http://localhost:5173
```

Backend runs on

```
http://localhost:5000
```
